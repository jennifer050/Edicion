## ðŸ¤– Antigravity (Asistente)

El repositorio de GitHub [Creaci-n-de-Contenidos](https://github.com/Miranda-Rayas006/Creaci-n-de-Contenidos) es un espacio de práctica de **Git y GitHub** que a su vez contiene varios **juegos interactivos educativos** diseñados para niños.

Esto es lo que contiene la estructura del repositorio:

### 1. Archivos en la raíz
*   **`Hola este es mi primer mensaje`** y **`Primer actividad`**: Archivos vacíos que probablemente se crearon como práctica inicial al aprender a usar comandos Git.

---

### 2. Directorio: `juego-numeros`
Es un juego didáctico interactivo de asociación diseñado para niños:
*   **Archivos**: `index.html`, `script.js` y `styles.css`.
*   **Propósito**: Permite a los niños relacionar números con la cantidad correspondiente de objetos (por ejemplo, manzanas) haciendo clic en ellos. Cuenta con una ventana modal de felicitación con estrellas al completar el juego.

---

### 3. Directorio: `practica-git`
Este directorio contiene ejercicios de práctica para comandos Git y dos juegos interactivos adicionales:
*   **`LEEME.md`**: Un archivo explicativo breve que detalla comandos básicos de Git (como `git init`, `git status`, `git add`, `git commit` y `git log`).
*   **`index.html` / `app.js` / `estilos.css`**: Una página básica de prueba con un título *"¡Hola Mundo!"* y un botón interactivo que muestra una alerta al hacer clic para verificar que el proyecto funciona correctamente.
*   **`Juego animales marinos con sonido.html`**: Un material didáctico interactivo de animales del mar diseñados para ser recortados. 
    *   Tiene un diseño optimizado para imprimirse en tamaño Carta o A4.
    *   Al hacer clic en cada animal (como el delfín 🐬, la tortuga 🐢 o el pulpo 🐙), el navegador utiliza la API de voz (`SpeechSynthesis`) para reproducir frases divertidas con voz infantil.
*   **`memorama-vocales.html`**: Un juego clásico de memoria (memorama) de vocales (A, E, I, O, U) para niños.
    *   Usa colores vibrantes y animaciones.
    *   Pronuncia las vocales en español con tono infantil usando la síntesis de voz al voltear las cartas.
    *   Incluye un contador de intentos y una animación de confeti al ganar.
*   **`capibara.png`**: Una imagen utilizada como recurso visual en esta sección.
